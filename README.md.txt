# 🧑‍🍳 Voice-Controlled Cooking Assistant

A voice-activated kitchen assistant built with Python that helps you cook by guiding you step-by-step through recipes, setting timers, playing YouTube videos, and providing nutritional and ingredient information — all through voice commands.

---

## 🚀 Features

- Start and follow recipe steps with your voice.
- Get ingredients and nutritional info for recipes.
- Play YouTube videos by saying “play [video/song name]”.
- Set kitchen timers (e.g., “set a timer for 2 minutes”).
- Change spice levels (mild/medium/spicy).
- Exit the assistant by saying "exit" or "stop".
